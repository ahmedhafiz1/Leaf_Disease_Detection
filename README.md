# Plant Disease Detection Using Deep Learning
Effective plant disease detection is essential for farmers to ensure healthy crops and maximize yields. This project leverages cutting-edge deep learning techniques to classify leaf images into 39 different categories, aiding in the early detection and management of plant diseases.

Key Features
Deep Learning Approach: The system utilizes a Convolutional Neural Network (CNN) built with the PyTorch framework to accurately classify leaf images.
Extensive Dataset: Training is conducted on the widely-used Plant Village Dataset, ensuring robustness and reliability.
Accessible and Easy to Use: The project is implemented as a Flask-based web application, allowing users to upload images and receive disease classifications quickly.
