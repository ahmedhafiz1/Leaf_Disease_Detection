About the Project

Revolutionizing Plant Health Management

Plant diseases can significantly impact agricultural productivity, making their detection crucial for every farmer. To address this need, we have developed a Plant Disease Detection System powered by Deep Learning. Our solution leverages a Convolutional Neural Network (CNN) to classify leaf images into 39 distinct categories with precision. The CNN model is implemented using the PyTorch Framework, ensuring robustness and scalability. For training, we utilize the comprehensive PlantVillage Dataset, which provides a diverse range of plant disease samples to enhance detection accuracy. This innovative approach aims to empower farmers with advanced tools to safeguard their crops effectively.

Plant Disease Detection Using Deep Learning

We utilize Deep Learning to classify plant diseases using leaf images. This project employs Convolutional Neural Networks (CNNs) to analyze visual data and achieve accurate predictions.

  • PyTorch-based model architecture.
  • Classification of 39 plant disease categories.
  • Extensive training with an augmented dataset.

Dataset Overview

The dataset contains 81,378 images across 39 classes, including healthy and diseased plant leaves.

The dataset comprises 81,378 images across 39 classes of plant leaves, including both healthy and diseased samples. To enhance the dataset, we applied six augmentation techniques: noise injection, image flipping, PCA color augmentation, gamma correction, rotation, and scaling.

Model Architecture

Our model is designed with layers of convolution and pooling, followed by fully connected layers.

  • Loss Function: CrossEntropyLoss combines softmax and categorical cross-entropy.
  • Optimizer: Adam for efficient training.
  • Achieved accuracies include 81% for training, 80% for validation, and 82% for testing..

Model Working

Detailed Flow of the Convolutional Neural Network (CNN) Model

The images are resized to 224x224 and fed into a CNN with 3 RGB channels. The first convolutional layer applies 32 filters, generating a feature map. This is followed by ReLU activation and Batch Normalization. The image then passes through a max pooling layer to reduce its size while retaining key features. This process is repeated through additional convolutional layers. Finally, the output is flattened and passed to a fully connected layer, predicting one of 39 categories. The final prediction is made by selecting the index with the highest value from the output tensor.
